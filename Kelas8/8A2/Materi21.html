Rata-rata (Mean)
Rata-rata adalah jumlah seluruh data dibagi banyaknya data.
Rata-rata = jumlah seluruh data / banyak data

Median
Median adalah nilai tengah dari data yang telah diurutkan.
Jika jumlah data ganjil, median adalah data di posisi tengah.
Jika jumlah data genap, median adalah rata-rata dari dua data tengah.

Modus
Modus adalah data yang paling sering muncul.
Contoh Soal 1: Data Ganjil
Data nilai ulangan: 70, 80, 90, 85, 75
Langkah:
Urutkan: 70, 75, 80, 85, 90
Rata-rata: (70 + 75 + 80 + 85 + 90) / 5 = 400 / 5 = 80
Median: data ke-3 = 80
Modus: tidak ada (semua muncul sekali)

Contoh Soal 2: Data Genap
Data tinggi badan (cm): 150, 155, 160, 165, 155, 160
Langkah:
Urutkan: 150, 155, 155, 160, 160, 165
Rata-rata: (150 + 155 + 155 + 160 + 160 + 165) / 6 = 945 / 6 = 157,5
Median: (155 + 160) / 2 = 157,5
Modus: 155 dan 160 (muncul dua kali)
