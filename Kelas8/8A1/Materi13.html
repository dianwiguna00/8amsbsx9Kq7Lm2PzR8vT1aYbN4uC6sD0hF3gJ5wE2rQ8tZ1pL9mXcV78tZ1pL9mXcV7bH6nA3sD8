1. Grafik Garis Lurus
Grafik dari persamaan garis lurus berbentuk garis yang memotong sumbu-x dan sumbu-y. Bentuk umum persamaan:
y = mx + c
- m = gradien (kemiringan garis)
- c = titik potong dengan sumbu-y

2. Langkah Menyajikan Grafik:
1. Tentukan dua titik dari persamaan (misalnya dengan memilih dua nilai x dan menghitung y).
2. Plot titik-titik tersebut di bidang koordinat.
3. Hubungkan titik-titik dengan garis lurus.

Contoh:
Persamaan: y = 2x + 1
- Jika x = 0 ⇒ y = 1 → titik (0,1)
- Jika x = 2 ⇒ y = 5 → titik (2,5)
- Hubungkan titik (0,1) dan (2,5) → garis lurus
Contoh Soal 1:
Gambarkan grafik dari persamaan y = -x + 3

Penyelesaian:
- Pilih dua nilai x:
x = 0 ⇒ y = 3 → titik (0,3)
x = 2 ⇒ y = 1 → titik (2,1)
- Plot titik (0,3) dan (2,1), lalu hubungkan dengan garis lurus.

Contoh Soal 2:
Gambarkan grafik dari y = ½x - 2

Penyelesaian:
- x = 0 ⇒ y = -2 → titik (0, -2)
- x = 4 ⇒ y = 0 → titik (4, 0)
- Plot titik (0, -2) dan (4, 0), lalu hubungkan.
